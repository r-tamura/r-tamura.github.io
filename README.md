# Installation

```
git clone https://github.com/r-tamura/r-tamura.github.io.git
npm i
```

# Scripts

```
// Generate HTML for publish
npm run build

// Clean directory for publish
npm run clean

// Run dev server(http://localhost:8000/)
npm run develop

// Deploy
npm run deploy
```
